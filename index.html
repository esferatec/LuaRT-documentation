Windows programming framework for Lua

Easy-to-use Lua programming environment for writing console or desktop applications for Windows

Features

Develop in Lua, a simple and fast programming language with a "batteries included" environment

Fast

Uses the latest Lua 5.4.5 VM

Powerful

Contains modules and objects for a wide variety of programming tasks.

Windows

Optimized runtime library for Windows

Compiler

Build standalone executables from your Lua scripts with embedded content

Lightweight

No dependencies or any other third party library

UTF8 Strings

Full support with patterns, substrings...

IDE

Write and debug code with Luart Studio

Develop in Lua

Develop only in Lua, no C/C++ compilation needed
